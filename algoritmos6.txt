USANDO LITSA - ARRAY E FOR

1 - FAÇA UM ALGORITMO QUE RECEBA UMA PALAVRA, OU FRASE
    - RETORNE ELA EM UM LISTA, SEPARADA POR LETRAS
    - EXEMPLO
        -> ENTRADA = "Palavra"
        -> SAIDA = ["P",'a','l','a','v','r','a']

2 - FAÇA UM ALGORITMO QUE RECEBA UM ARRAY DE PALAVRAS
    - RETORNE AS PALAVRAS EM UMA STRING, OBEDECENDO OS ESPAÇOS ENTRE AS PALAVRAS
    - EXEMPLO 
        -> ENTRADA = ['Amanhã','sexta','feira','feriado']
        -> SAIDA = "Amanhã sexta feira feriado"

3 - FAÇA UM ALGORITMO QUE RECEBA UM ARRAY DE NUMEROS
    - RETORNE DOIS NUMEROS, O MAIOR E O MENOR ENTRE ELES
    - EXEMPLO 
       -> ENTRADA = [0,3,4,6,7,9]
       -> SAIDA = [0,9]
