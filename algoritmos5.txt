USANDO LITSA - ARRAY

1 - DADO UM ARRAY DE NÚMEROS QUALQUER, 
    - RETORNE UM NOVO ARRAY COM OS VALORES MULTIPLICADOS PELO INDEX

2 - DADO UM ARRAY DE PALAVRAS QUALQUER,
    - RETORNE UM NOVO ARRAY COM A ORDEM DOS NOMES AO CONTRARIO
    - USANDO FOR

3 - DADO UM ARRAY DE PALAVRAS QUALQUER
    - RETORNE UM NOVO ARRAY COM AS PALAVRAS QUE COMEÇAM COM A LETRA "a"
    - SE NÃO HOUVER NEHUMA, RETORNE UM ARRAY VAZIO

4 - DADO UM ARRAY DE NUMEROS QUALQUER
    - RETORNE UM NOVO ARRAY, NO SEGUINTE FORMATO,
    - SE NUMERO IMPAR - "impar"
    - SE NUMERO PAR - "par"
    - SE NUMERO MULTIPLO DE 5 - "xazam"

5 - DADO UM ARRAY DE NUMEROS QUALQUER
    - RETORNE A SOMA DE TODOS OS NUMEROS
